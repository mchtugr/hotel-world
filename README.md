## Run locally

Clone the project

```bash
  git clone https://github.com/mchtugr/hotel-world.git
```

Go to project directory

```bash
  cd hotel-world
```

Install dependencies

```bash
  npm install
```

Run the app

```bash
  npm run serve
```

## Built with

- VueJs
- Vue Bootstrap
- Vuelidate
- Vue Router
